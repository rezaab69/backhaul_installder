# backhaul_installder
  A powerful, full-featured menu-driven installer/manager for the   Musixal/Backhaul reverse-tunneling tool on systemd-based Linux systems   (Debian, Ubuntu and derivatives).


# Download and run
chmod +x backhaul-manager.sh
sudo ./backhaul-manager.sh

# Or with direct commands:
sudo ./backhaul-manager.sh install
sudo ./backhaul-manager.sh diag
